Python Data Structures Multiple Choice Questions

Lists

  1. Which method is used to add an element to the end of a list?
    a) list.add()
    b) list.append()
    c) list.insert()
    d) list.extend()
  2. What is the output of [1, 2, 3] + [4, 5, 6]?
    a) [1, 2, 3, 4, 5, 6]
    b) [[1, 2, 3], [4, 5, 6]]
    c) [5, 7, 9]
    d) Error
  3. How do you remove the first occurrence of value 'x' from list 'my_list'?
    a) my_list.remove('x')
    b) my_list.pop('x')
    c) my_list.delete('x')
    d) del my_list['x']
  4. What is the time complexity of accessing an element in a list by its index?
    a) O(1)
    b) O(n)
    c) O(log n)
    d) O(n^2)
  5. Which of the following will sort a list 'my_list' in descending order?
    a) my_list.sort(reverse=True)
    b) sorted(my_list, reverse=True)
    c) my_list.reverse()
    d) Both a and b

Dictionaries

  1. How do you create a dictionary with keys 'a', 'b', 'c' all with value 0?
    a) dict('a'=0, 'b'=0, 'c'=0)
    b) {'a': 0, 'b': 0, 'c': 0}
    c) dict.fromkeys(['a', 'b', 'c'], 0)
    d) Both b and c
  2. What happens if you try to access a key that doesn't exist in a dictionary?
    a) Returns None
    b) Raises a KeyError
    c) Returns an empty string
    d) Creates a new key-value pair with None as the value
  3. Which method removes all items from a dictionary?
    a) dict.remove_all()
    b) dict.clear()
    c) dict.delete()
    d) dict = {}
  4. What is the output of list({"a": 1, "b": 2, "c": 3}.keys())?
    a) ["a", "b", "c"]
    b) [1, 2, 3]
    c) [("a", 1), ("b", 2), ("c", 3)]
    d) The order is not guaranteed
  5. How can you merge two dictionaries in Python 3.9+?
    a) dict1.update(dict2)
    b) dict1 | dict2
    c) dict1.merge(dict2)
    d) Both a and b

Sets

  1. Which of the following is not a valid way to create an empty set?
    a) set()
    b) {}
    c) set([])
    d) frozenset()
  2. What is the output of {1, 2, 3} & {3, 4, 5}?
    a) {1, 2, 3, 4, 5}
    b) {3}
    c) {1, 2, 4, 5}
    d) {3, 3}
  3. Which method removes an element from a set without raising an error if the element is not present?
    a) remove()
    b) discard()
    c) pop()
    d) delete()
  4. What is the time complexity of checking if an element is in a set?
    a) O(1) on average
    b) O(n)
    c) O(log n)
    d) O(n^2)
  5. Which of the following is true about sets in Python?
    a) They can contain duplicate elements
    b) They are ordered
    c) They can contain mutable elements like lists
    d) They are unordered and do not allow duplicates

    a) (1)
    b) (1,)
    c) 1,
    d) Both b and c
  2. What is the output of (1, 2, 3) + (4, 5, 6)?
    a) (1, 2, 3, 4, 5, 6)
    b) ((1, 2, 3), (4, 5, 6))
    c) (5, 7, 9)
    d) Error
  3. Which of the following is true about tuples?
    a) They are mutable
    b) They can be used as dictionary keys
    c) They cannot contain mutable objects
    d) They cannot be nested
  4. What is the correct way to unpack the tuple (1, 2, 3) into three variables?
    a) a, b, c = (1, 2, 3)
    b) a = (1, 2, 3)[0]; b = (1, 2, 3)[1]; c = (1, 2, 3)[2]
    c) a, b, c = 1, 2, 3
    d) Both a and c
  5. Which method returns the number of times a value appears in a tuple?
    a) tuple.count()
    b) tuple.index()
    c) tuple.find()
    d) tuple.occurrences()
